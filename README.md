# Phishing Awareness Training System (GUI)

This project is a Python-based GUI application developed using Tkinter.
It is designed to educate users about phishing attacks, how to identify them,
and best practices to stay safe online.

## Features
- Introduction to Phishing
- Identify Phishing Emails
- Fake Website Detection
- Social Engineering Techniques
- Best Practices for Security
- Simple Quiz Module

## Technologies Used
- Python
- Tkinter (GUI)

## How to Run the Project
1. Install Python (3.x recommended)
2. Download or clone this repository
3. Open terminal in project folder
4. Run the command:

   python phishing_awareness.py
